Pipeline Samples

Pipelines are used to modify and enrich your data while it is being indexed into Elasticsearch. In order to build a pipeline, we need to define at least one Sample. A Sample is an example of data that your pipeline will be expected to work with. You can click 'Add Single Sample' to enter or paste a single JSON object. You can also click 'Add Samples from Logs' to create multiple samples at one time from line delimited text.
